Projekt Memory Game

Strona główna Wersja jQuery Wersja p5js

Projekt gry Memory Game wykonany przez uczniów klasy 1a Collegium Gostomianum, w składzie:

      Michał Rawski
      Adam Garnuszek
      Wiktor Malinowski

Gra polega na tym, by w jak najmniejszej liczbie ruchów dopasować do siebie dwie karty z takimi samymi obrazkami / kolorami, "klikając" na nie. Jeśli obrazki do siebie nie pasują, obie karty zakrywają się spowrotem.
Aby zagrać w grę, wybierz jedną z jej wersji w menu powyżej: wersja jQuery - polegająca na znalezienu takich samych obrazków z owocami lub wersja p5 - polegająca na znalezienu takich samych kolorów.
